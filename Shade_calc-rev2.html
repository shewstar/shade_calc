ELI5: How the two error checks work

1) Percent Error (angle closure check)

Think of joining sticks to make a loop. If the shape is perfect, all inside corner angles should add up to a known total. For 4 points that total is 360 degrees. For 5 points that total is 540 degrees. The app compares your calculated angle sum to the expected total. The gap is shown as percent error.

Expected corner sum = (points - 2) * 180 4-point: A ----- B | | D ----- C Expected: A+B+C+D = 360 degrees Percent Error = |actual sum - expected sum| / expected sum * 100

2) Residual Fit (distance mismatch check)

Imagine placing pins for corners so all measured side and diagonal distances match as closely as possible. For each measurement, the app checks how far off that best-fit shape is. RMS is the average mismatch size in mm (lower is better).

Measured distances ---> Best-fit shape ---> Mismatch per line Example line mismatch: Measured A-C = 5650 mm Best-fit A-C = 5632 mm Residual = -18 mm RMS mismatch = "typical" residual size across all lines (in mm)

Why they can disagree: angle percent is strict about corner-angle math (best for convex layouts), while residual fit is strict about distance consistency (works better with internal/reflex corners too).
